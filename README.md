# MalinkaHelp_Bot

## Установка

### linux
```
sudo apt install python3 # Debian/Ubuntu
sudo pacman -S python    # Arch Linux
git clone https://github.com/d0m-1k/MalinkaHelp_Bot.git
cd MalinkaHelp_Bot
python3 -m venv venv
```
